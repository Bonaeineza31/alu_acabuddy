**ALU AcaBuddy**

**Project Description**

ALU AcaBuddy is a Flutter mobile app created to help students manage their academic life more easily. The app brings together important student tools like assignments, attendance, and session scheduling in one place so that students can stay organized and manage their time better.

This project was built as part of a learning experience in mobile app development using Flutter. The goal was to design and develop a simple but useful academic assistant for students.

**Features ** 
User onboarding and authentication

Home dashboard for quick overview

Session scheduling

Assignment and task tracking

Attendance tracking

Clean and easy-to-use interface

Data maintained during current session only

**Tools Used**

Flutter

Dart

Provider (state management)

Material UI

**How to Run the Project**

Clone the repository

git clone

Open the project folder

cd alu_acabuddy

**Install dependencies**

flutter pub get

Run the app

flutter run

Make sure Flutter and an emulator or physical device are installed before running the project.
